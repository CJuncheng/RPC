
# buttonrpc - modern rpc framework for C++
- ZeroMQ 作为网络层
- 使用c++14开发

## Features
- 轻量级，跨平台，简单易用
- 服务端可以绑定自由函数，类成员函数，std::function对象
- 服务端可以绑定参数是任意自定义类型的函数
- 客户端与服务端自动重连机制
- 客户端调用超时选项


## Dependences
- [ZeroMQ](http://zguide.zeromq.org/page:all)


## Building
- windows vs2015 或者更高版本,  linux 添加编译选项：-std=c++1z
